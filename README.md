# Projeto Saúde e Treino

Projeto final da UC de back-end - Integração entre front-end e back-end

## Sobre o projeto

O projeto tem como objetivo simular um aplicativo de academia, onde possui landing page divulgando a marca e os serviços, com feedbacks fictícios de personas que já utilizaram e deixaram sua opinião. A aplicação possui tela de login e cadastro de usuários funcionais para o usuário conseguir posteriormente acessar a tela home com os exercícios distribuidos em abas diferentes.

## Como Executar o Projeto

Siga os passos abaixo para executar o projeto localmente.

1. Clonar o Repositório

```
git clone https://github.com/guilhermevaz4658/projeto-final
```

2. Entrar na pasta do projeto

```
cd nome-do-repositorio
```

3. Configurar as Variáveis de Ambiente para conexão com banco de dados

4. Criar o Banco de Dados

5. Entrar na pasta back-end e instalar as dependências:

```
cd back-end
npm install
```

6. Entrar na pasta front-end e instalar as dependências:

```
cd ..
cd front-end
npm install
```

7. Rodar o Servidor Back-end:

```
cd .. 
cd back-end
npm run dev
```

8. Abrir outro terminal e rodar o Front-End:

```
cd front-end
npm run dev
```

9. Acessar a porta disponibilizada no front-end:

Provavelmente:
```
http://localhost:5173/
```
