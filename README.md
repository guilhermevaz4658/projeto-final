# Projeto Saúde e Treino

Projeto final da UC de back-end - Integração entre front-end e back-end

## Sobre o projeto

O projeto tem como objetivo simular um aplicativo de academia, onde possui landing page divulgando a marca e os serviços, com feedbacks fictícios de personas que já utilizaram e deixaram sua opinião. A aplicação possui tela de login e cadastro de usuários funcionais para o usuário conseguir posteriormente acessar a tela home com os exercícios distribuidos em abas diferentes.
